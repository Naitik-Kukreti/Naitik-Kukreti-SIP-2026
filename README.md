# Yatra Sathi

## Homestay & Eco-Tourism Platform

Yatra Sathi is a simple and responsive web application developed using HTML, CSS, and JavaScript. The platform is designed to promote homestays, eco-tourism services, and community-led accommodations while providing travelers with an easy way to explore authentic travel experiences.

---

## Project Description

The website serves as a frontend prototype for a homestay and eco-tourism platform. Users can explore featured homestays, learn about the platform, view dashboard information, and access a login interface.

This project was developed as part of the **AI-Assisted Full Stack Web Development Internship Program (Week 2 – Frontend Foundations)**.

---

## Features

* Responsive Navigation Bar
* Hero Section with Call-to-Action Button
* Featured Homestay Cards
* About Section
* Dashboard Section
* Login Form
