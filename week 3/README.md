# Yatra Sathi

Yatra Sathi is an AI-powered eco-tourism and homestay recommendation website for travelers exploring Uttarakhand and similar sustainable tourism destinations.
## Week 3 Deliverables Covered

- Reusable component library in `src/components/ui/`.
- UI components: Button, Input, Modal, Toast, and Loader.
- `src/components/ui/index.js` exports all UI components.
- Components are used on the Showcase, Login, and AI Planner pages.
- Dark/light mode toggle persists using `localStorage`.

## Run Locally

```bash
python -m http.server 5173
```

Then open:

```text
http://localhost:5173
```
